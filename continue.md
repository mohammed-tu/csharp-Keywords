# continue

تستخدم لتخطي الاوامر اللتي تليها واستكمال مربع التكرار 

**مثال**


<div dir="ltr">


```
            for (int i = 0; i < 7; i++) // طباعة الأرقام من 0 غلى 6
            {
                if (i == 4)
                    continue; // عند الوصول للرقم 4 يتم تخطي الاوامر واستكمال مربع التكرار
                Console.WriteLine(i);

            }
```
