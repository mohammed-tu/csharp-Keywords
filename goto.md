# goto 

Keyword للتنقل لأجزاء معينة من البرنامج ، بحيث يكون الجزء المرادالانتقال له معرف مسبقا بالكما هو موضح بالمثال

**مثال**

يقوم المثال التالي بطباعة نص ثلاث مرات بإستخدام goto


<div dir="ltr">


```

            int i = 0;
        lable1:
            Console.WriteLine("hi");
            i++;
            if (i < 3)
                goto lable1;
```
