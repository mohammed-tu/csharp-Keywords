# break 

هي keyword من نوع jump statement تستخدم للخروج من مربع التكرار 

**مثال**

<div dir="ltr">
  
```
            for (int i = 0; i < 7; i++) // طباعة الأرقام من 0 غلى 6
            {
                Console.WriteLine(i);
                if (i == 3)
                    break; // بعد طباعة الرقم 3 يتم الخروج من مربع التكرار
            }

```
